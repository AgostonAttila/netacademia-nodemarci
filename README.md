# Netacademia Node Marci! tanfolyam

Marci egy becsületes, rendes C# programozó. A képzés moduljain végighaladva be fogjuk neki bizonyítani, hogy a .NET-en kívül is van élet. JavaScript segítségével érünk el fájlokat, csatlakozunk adatbázishoz, és más huncutságokat is el fogunk még követni. A tabuk azért vannak, hogy ledöntsük õket. Az utolsó modul végére Marci megszereti a Node.js-t. Marci okos. Légy olyan, mint Marci!

## A Node.js fejlesztõkörnyezet telepítése és beállítása
A tanfolyam elsõ moduljában telepítjük a fejlesztõkörnyezetet, megismerkedünk a parancssorral, és elkészítjük az elsõ szerverünket Node.js segítségével.

## A modulok és az események megismerése
Megismerkedünk a Node.js modulokkal, elkészítjük a saját modulunkat, és kipróbáljuk azt. Mindemellett bemutatjuk az egyes események létrehozását, kezelését, és megismerkedünk az egyedi eseményekkel is.

## A fájlok kezelése
Bemutatjuk, hogyan lehetséges a fájlok kiszolgálása a felhasználók felé (írás, olvasás, mappakezelés), és elkezdünk megismerkedni az Express névre hallgató Node.js keretrendszerrel.
Az Express megismerése
Ebben a modulban mélyebben megvizsgáljuk az Expresst, megismerkedünk a sablonok rendszerével, és számos tippet, trükköt fogunk bemutatni az elsõ lépések sikeres megtételéhez.
